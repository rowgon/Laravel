# Laravel Static Site

## Installation
1. Make sure you have PHP, Composer, and Laravel installed on your machine.
2. Clone this project or extract it into your local development environment.
3. Run `composer install` to install dependencies.
4. Run `php artisan serve` to start the local server.

## Features
- Static pages: Home, About, Services, Contact.
- Simple Bootstrap design.
- Functional contact form (requires Mailtrap for email testing).

Enjoy exploring and customizing the project!
